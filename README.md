## 📱 About The Project

**Axorvian Store** is a professional, feature-rich e-commerce mobile application built with Flutter. Developed as part of the **Axorvian Internship Program**, this app demonstrates modern mobile development practices with a focus on clean architecture, responsive UI, and seamless user experience.

The app provides a complete online shopping experience including user authentication, dynamic product listing, shopping cart management, wishlist, and order processing.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔐 **Authentication** | Login & Signup with local storage |
| 🏠 **Splash Screen** | Beautiful animated splash screen |
| 📦 **Product Listing** | Dynamic product grid with images, prices, ratings |
| 🔍 **Search** | Real-time product search functionality |
| 🏷️ **Category Filter** | Browse products by categories |
| 📄 **Product Details** | Detailed view with images and description |
| 🛒 **Shopping Cart** | Add/Remove items, update quantities |
| ❤️ **Wishlist** | Save favorite products |
| 👤 **User Profile** | Personal information management |
| 📋 **Order Summary** | Checkout with payment method selection |
| 📊 **Order History** | Track all placed orders |
| 📱 **Responsive UI** | Adapts to all screen sizes |

## 🚀 Getting Started

### Prerequisites

- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0
- Android Studio / VS Code
- Android SDK (for Android development)

### Installation

```bash
# Clone the repository
git clone https://github.com/Muhammadwaleed12743/axorvian_store.git

# Navigate to project directory
cd axorvian_store

# Install dependencies
flutter pub get

# Run the app
flutter run
